# vue-use-vuetify

```
프로젝트 명 : vuetify 를 적용한 vue admin 프로젝트
목표 : vue 공부를 통해 차기 프로젝트 러닝커브 단축
```
## Project setup
```
해당 프로젝트는 각 모듈 및 라이브러리 관리를 위해 npm을 사용한다.

// 모듈 설치
npm install

// Compiles and hot-reloads for development
npm run serve

// Compiles and minifies for production
npm run build

// Run your tests
npm run test

// Lints and fixes files
npm run lint
```
